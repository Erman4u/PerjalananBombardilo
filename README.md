# Perjalanan Bombardilo

Perjalanan Bombardilo adalah sebuah mini game web berbasis _endless runner_ bergaya retro. Bantu Bombardilo sang buaya untuk melompat sejauh mungkin melintasi rintangan awan!

## Fitur
- **Responsive Design**: Bisa dimainkan di komputer (Desktop) maupun di HP (Mobile).
- **Difficulty Levels**: Tersedia tingkat kesulitan Easy, Normal, dan Hard yang mempengaruhi kecepatan, gravitasi, dan frekuensi rintangan.
- **Dynamic Scoring & Background**: Skor dihitung berdasarkan waktu bertahan (jarak), dan latar belakang akan berganti setiap mencapai kelipatan skor tertentu (1000 poin).
- **Music & Sound**: Dilengkapi dengan background music yang dapat di-mute/unmute melalui tombol di pojok kanan atas.

## Cara Bermain
### Di Komputer / Laptop
- **Spasi / Klik Kiri**: Tekan tombol spasi atau klik kiri pada mouse untuk memulai permainan dan membuat Bombardilo melompat.

### Di HP / Tablet
- **Tap Layar**: Ketuk layar untuk memulai dan melompat.

## Teknologi yang Digunakan
- **HTML5 Canvas**: Untuk proses rendering game.
- **CSS3**: Untuk styling dan membuat tampilan responsif pada UI.
- **Vanilla JavaScript**: Untuk logika utama game.

## Menjalankan Game Secara Lokal
1. Clone repositori ini atau download sebagai ZIP.
2. Ekstrak file jika mendownload ZIP.
3. Buka file `index.html` menggunakan web browser favorit Anda (Chrome, Firefox, Safari, dll).
